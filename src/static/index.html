What is Maqam?
Understanding Arabic Modal Music

Maqam (ŸÖŸÇÿßŸÖ) is the system of melodic modes used in Arabic music. Each maqam defines a unique set of notes, intervals, and melodic patterns that give it its distinctive character.

A maqam is built from smaller building blocks called Jins (ÿ¨ŸÜÿ≥) - typically a lower tetrachord and an upper tetrachord combined together.

This detector uses 36-bin microtonal analysis to capture the quarter-tones and neutral intervals unique to Arabic music.

How It Works
Technical Overview
  1. Audio Processing: HPSS isolates the melodic content
  2. Pitch Analysis: CQT with 36 bins per octave captures microtones
  3. Tonic Detection: Finds the rukooz (tonic note)
  4. Jins Matching: Identifies the two constituent tetrachords
  5. Maqam Prediction: Markov chains + Neural network classify the maqam
